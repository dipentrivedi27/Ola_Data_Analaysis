# 🚖 OLA Ride Analytics Dashboard (SQL & Power BI)

## 📌 Project Overview
This project focuses on analyzing ride-booking data from an OLA-like platform using SQL and Power BI. The goal is to uncover insights related to booking trends, cancellations, customer behavior, and revenue performance to improve operational efficiency and service quality.

---

## 🎯 Problem Statement
The ride-booking platform lacked a clear understanding of:
- Booking trends  
- Ride cancellations  
- Customer and driver behavior  
- Revenue performance across vehicle types  

This made it difficult to optimize operations and enhance service quality.

---

## 🎯 Objectives
- Analyze ride booking patterns  
- Identify key reasons for ride cancellations  
- Understand customer and driver behavior  
- Evaluate revenue performance across vehicle types  
- Build interactive dashboards for decision-making  

---

## 🗂️ Dataset Description
The dataset contains 20,000+ ride records, including:
- Ride ID and booking details  
- Customer and driver information  
- Vehicle type  
- Ride status (Completed, Cancelled, etc.)  
- Payment method  
- Ratings and feedback  
- Date and time of bookings  

---

## 🛠️ Tools & Technologies
- **SQL (MySQL)** – Data extraction and analysis  
- **Power BI** – Data visualization and dashboard creation  

### SQL Concepts Used:
- SELECT  
- JOIN  
- GROUP BY  
- ORDER BY  
- Aggregate Functions (SUM, COUNT, AVG)  

---

## 📊 Analysis & Dashboard Features

### 🔹 Ride Analysis
- Total ride volume  
- Booking status breakdown (Completed, Cancelled)  
- Ride trends over time  

---

### 🔹 Cancellation Insights
- Cancellation rate analysis  
- Reasons for ride cancellations  
- Customer vs driver cancellations  

---

### 🔹 Customer & Driver Insights
- Top customers based on ride frequency  
- Top drivers based on performance  
- Customer and driver ratings analysis  

---

### 🔹 Revenue Analysis
- Total revenue generated  
- Revenue distribution by vehicle type  
- High-performing vehicle categories  

---

### 🔹 Payment Insights
- Payment method distribution (Cash, Online, Wallet, etc.)  
- Most preferred payment modes  

---

## 📈 Key Insights
- Identified peak booking hours and high-demand periods  
- Discovered major reasons for ride cancellations  
- Found top-performing drivers and loyal customers  
- Highlighted revenue contribution by different vehicle types  
- Improved understanding of customer preferences  

---

## 🚀 How to Use
1. Import dataset into MySQL  
2. Perform data cleaning and analysis using SQL queries  
3. Connect Power BI to the dataset  
4. Build interactive dashboards  
5. Analyze and interpret insights
